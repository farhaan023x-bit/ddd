# Spandan Dashboard (DDD)

## Overview
Teacher Dashboard for Student Performance Monitoring.

This feature provides insights into student engagement, performance, and question analytics during sessions.

---

## Features
- Session Overview
- Student Performance Tracking
- Question Analytics
- Activity Monitoring

---

## Tech Stack
- Frontend: React
- Backend: Node.js, Express
- Testing: Thunder Client / Browser

---

## Backend APIs
- GET /student-performance
- GET /questions/:id
- POST /activity
