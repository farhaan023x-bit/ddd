# 🚀 DDD – Teacher Dashboard (Spandan)

## 📌 Project Overview

DDD (Dopamine Driven Dashboard) is a performance and analytics dashboard designed to provide real-time insights into student engagement, performance, and learning trends.

This implementation focuses on the **Teacher Dashboard for Student Performance Monitoring**, enabling instructors to analyze session data efficiently.

---

## 🎯 Implemented Feature

### 👩‍🏫 Teacher Dashboard for Student Performance Monitoring

This feature provides a centralized dashboard for teachers to monitor student activity and performance during and after a session.

---

## ⚙️ Functionalities Implemented

### 🔹 Session Overview
- Session status (live/completed)
- Total students
- Total questions
- Total points distributed

### 🔹 Student Performance
- Questions attempted
- Correct / Incorrect answers
- Total points
- Average response time
- Sorting by points

### 🔹 Question Analytics
- Response count per question
- Correct answer percentage
- Average answer time
- Difficulty indication

### 🔹 Points Insights
- Average points
- Highest scoring student
- Lowest scoring student

### 🔹 Achievements Monitoring
- Badge display for students

### 🔹 Post Session Summary
- Total participation
- Overall performance summary

### 🔹 Real-Time Simulation
- Auto-refresh every 5 seconds
